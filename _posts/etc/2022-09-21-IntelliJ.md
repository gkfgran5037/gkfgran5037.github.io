## 코드 수정 단축키
<br/>

| KEY1  | KEY2  | KEY3 | 실행 내용                              |
|-------|-------|------|------------------------------------|
| Ctrl  | D     |      | 커서 위치한 라인 복제                       |
| Ctrl  | Y     |      | 커서 위치한 라인 삭제                       |
| Ctrl  | Alt   | M    | 메소드 추출                             |
| Alt   | Shift | R    | 변수명 일괄 변경                          |
| Shift | F6    |      | 변수명 일괄 변경                          |
| Alt   | Ins   |      | 생성자, Getter, Setter, 테스트 케이스 자동 생성 |

<br/><br/><br/>
